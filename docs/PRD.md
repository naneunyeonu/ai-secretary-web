# 1인 프로젝트: AI 투자 비서
## 프로젝트 개요
* **프로젝트명** : AI Secretary Web
* **목표** : 개인 투자자의 정보 수집시간을 단축시켜주는 대시보드 구축
* **타겟 사용자** : 데이터 기반의 투자를 지향하는 개인 투자자, 정보를 찾는 시간을 줄이고자 하는 개인 투자자

## 핵심 기능 정의
### 개발 완료 기능
* **포트폴리오 관리**
    * 보유 종목 등록 및 실시간 평가금액, 수익률 조회
* **대시보드 관리**
    * 관심 종목 등록 및 해당 종목에 대한 AI 브리핑

### 개발 예정 기능
* **현 시점 주도 섹터 및 주도주 추출기**
    * 당일 기준 1개월 상승률 30위 종목을 분석, 주도 테마 파악(국장, 미장 분리 산출)
* **경제 캘린더**
    * 경제 지표 발표일, 관심 기업 실적 발표일 등을 캘린더에 표시
* **수급 추적기**
    * 코스피에 대한 주체별(개인, 외국인, 기관) 순매수, 순배도 동향 시각화. 특히 외국인/기관의 'N일 연속 순매수' 종목을 자동 필터링하여 제공
